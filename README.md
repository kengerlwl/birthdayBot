# birthdaybot

github action 生日提醒


# 使用

在`config`文件夹中

`config.yaml`文件配置自己的待提醒的邮箱

`people.yaml`文件配置生日提醒列表以及具体参数。具体可以参考demo


目前设置是最多提前三天提醒

# 关于加密
如果不想写邮箱的一些关键验证信息，那么就建议存入github action的加密变量里面。secrets。
